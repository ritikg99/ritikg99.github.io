Implementation of the "12 Steps to Navier-Stokes Equations" by Prof. Lorena A. Barba, originally developed in Python, ported to C++ as a hands-on introduction to scientific computing fundamentals.

View C++ Implementation on GitHub

Key Aspects Covered:

  • Step-by-step progression from basic diffusion to full Navier-Stokes solutions
  • Implemented fundamental numerical methods for fluid flow simulation
  • Solved both cavity flow and channel flow problems
  • Developed versions in both C++ and Python for comparison

Technical Implementation:

  • Core C++ concepts: arrays, pointers, function definitions
  • Spatial and temporal discretization techniques
  • Implementation of various boundary conditions
  • Finite difference methods for PDE solutions
  • Visualization of velocity and pressure fields

Learning Outcomes:

  • Deeper understanding of CFD mathematical foundations
  • Practical experience with numerical methods implementation
  • Comparison of Python vs. C++ for scientific computing
  • Hands-on experience with stability criteria and convergence
  • Application to classic benchmark problems (cavity/channel flows)
